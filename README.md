#  Avaliação Técnica - Dados Jurídicos

##  Objetivo
Este repositório contém a resolução de uma avaliação técnica com foco em análise de dados, modelagem e manipulação de informações jurídicas.

---

##  Tecnologias utilizadas
- Python (Pandas)
- SQL
- Git/GitHub

---

##  Atividades desenvolvidas
- Construção de consultas SQL com JOIN e filtros
- Modelagem de dados relacional (SQLAlchemy) e não relacional (JSON/Firestore)
- Tratamento e limpeza de dados com Pandas
- Estruturação de dados para análise
- Implementação básica de Web Scraping

---

##  Dificuldades encontradas
- Ajustes em consultas SQL, principalmente em JOIN e filtros por data  
- Tratamento de dados inconsistentes (ex: valores monetários em formatos diferentes)  
- Entendimento inicial de modelagem NoSQL  
- Web Scraping em páginas com carregamento dinâmico  
- Configuração e uso do Git para versionamento  

---
